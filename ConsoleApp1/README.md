Лабораторная работа: Разработка приложения Class Enrolment.

Цель: Разработать консольное приложение на C# для управления записью студентов на курсы.
Интерфейс:
·  Реализовать текстовое меню для взаимодействия с пользователем.
Меню: 
1. Добавить курс
2. Посмотреть курс
3. Удалить курс
4. Записать студента на курс
5. Показать список студентов на курсе
6. Удалить студента из курса
7. Выход

1.  Реализация функционала:
·  Добавить возможность добавления/просмотра/удаления нового курса.
·  Добавить возможность записи студента на курс (проверка на наличие свободных мест).
·  Добавить возможность вывода списка студентов, записанных на конкретный курс.
·  Добавить возможность удаления студента из кур


Используются только простые типы данных и методы
